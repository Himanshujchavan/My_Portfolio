## 📦 Installation

### Clone the Repository

```bash
git clone https://github.com/Himanshujchavan/My_Portfolio.git
```

### Navigate to the Project

```bash
cd My_Portfolio
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```

in your browser.
